# enzymepromiscuity


This repository contains the model described in "Frustration can limit the adaptation of
promiscuity enzymes through gene duplication and specialisation" 
(Schmutzer, Dasmeh and Wagner, 2023). All code is written for python3. The files and their contents are:

promiscuity.py           Optimisation model and figure plotting
surfaceplot.py           Functions for creating the surface plot in figure 4
test_promiscuity.py      Unit-tests of the optimisation model
